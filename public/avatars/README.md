# Avatars

This folder contains a set of 10 lightweight SVG avatars used by the Profile avatar picker.

- You can replace these files with your own images (SVG, PNG, JPG, WEBP).
- Keep the filenames consistent (avatar-1.svg … avatar-10.svg) or update the `presetAvatars` array in `features/admin/pages/settings/pages/profile/form.tsx` accordingly.
- Recommended size: 128x128 or higher with a square aspect ratio.
